1. The DOM (Document Object Model) represents the document as nodes and objects; that way, programming languages can interact with the page.

3. If you add HTML to a page using innerHTML (or several jQuery methods), you need to be aware of Cross-Site Scripting Attacks (XSS) otherwise, an attacker could gain access to your users' accounts.
